# DB-Engine


DB mini engine for Database II course that supports B+ tree Index

*Features:*
1. **Table Operations:**
   - Creation of Tables
   - Deletion of Tables
   - Insertion into Tables
   - Deletion from Tables
   - Selection from Tables
     
2. **Index Operations:**
   - Creation of Indexes

3. **SQL Parsing:**
   - Utilization of ANTLR
   - Implementation of a Self written Grammar
